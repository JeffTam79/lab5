### HTML Tag Cheat Sheet
| Syntax | Description | Example |
|--------|------------|---------|
| `<style></style>` | A tag used to define CSS styles within an HTML document. | ```html
<style>
    body {
        background-color: lightgray;
    }
</style>
``` |
| `font-family` | Specifies the font of the text. | ```css
p {
    font-family: Arial, sans-serif;
}
``` |
| `font-size` | Defines the size of the font. | ```css
h1 {
    font-size: 24px;
}
``` |
| `font-weight` | Determines the thickness of the font. | ```css
strong {
    font-weight: bold;
}
``` |
| `font-style` | Defines the style of the font (normal, italic, oblique). | ```css
em {
    font-style: italic;
}
``` |
| `font-variant` | Controls text appearance, such as small caps. | ```css
p {
    font-variant: small-caps;
}
``` |
| `font` | Shorthand for setting font-related properties. | ```css
p {
    font: italic bold 16px Arial, sans-serif;
}
``` |
| `text-align` | Aligns text to left, right, center, or justify. | ```css
h1 {
    text-align: center;
}
``` |
| `text-decoration` | Controls text underline, overline, and line-through. | ```css
a {
    text-decoration: none;
}
``` |
| `text-transform` | Changes the capitalization of text. | ```css
p {
    text-transform: uppercase;
}
``` |
| `text-indent` | Indents the first line of a paragraph. | ```css
p {
    text-indent: 20px;
}
``` |
| `margin-top` | Sets the top margin of an element. | ```css
div {
    margin-top: 10px;
}
``` |
| `margin-right` | Sets the right margin of an element. | ```css
div {
    margin-right: 10px;
}
``` |
| `margin-bottom` | Sets the bottom margin of an element. | ```css
div {
    margin-bottom: 10px;
}
``` |
| `margin-left` | Sets the left margin of an element. | ```css
div {
    margin-left: 10px;
}
``` |
| `max-width` | Sets the maximum width an element can be. | ```css
div {
    max-width: 600px;
}
``` |
| `padding` | Sets the space between an elementâ€™s content and border. | ```css
div {
    padding: 10px;
}
``` |
| `margin` | Shorthand property for setting margins. | ```css
div {
    margin: 10px 20px;
}
``` |
| `display: flex` | Defines a flex container. | ```css
div {
    display: flex;
}
``` |
| `border-bottom` | Sets the bottom border of an element. | ```css
h1 {
    border-bottom: 2px solid black;
}
``` |
| `border-top` | Sets the top border of an element. | ```css
h1 {
    border-top: 2px solid black;
}
``` |
| `border-radius` | Rounds the corners of an element. | ```css
div {
    border-radius: 10px;
}
``` |
| `background-color` | Sets the background color of an element. | ```css
body {
    background-color: lightblue;
}
``` |
| `display` | Determines how an element is displayed. | ```css
div {
    display: inline-block;
}
``` |
| `list-style-type` | Defines the appearance of list bullets or numbers. | ```css
ul {
    list-style-type: square;
}
``` |
| `pseudo-classes` | Used to define a special state of an element. | ```css
a:hover {
    color: red;
}
``` |
| `nav a:link` | Styles unvisited links in a navigation bar. | ```css
nav a:link {
    color: blue;
}
``` |
| `nav a:visited` | Styles visited links in a navigation bar. | ```css
nav a:visited {
    color: purple;
}
``` |
| `nav a:hover` | Changes the style when the mouse hovers over a link. | ```css
nav a:hover {
    color: red;
}
``` |
| `nav a:active` | Styles the link when it is clicked. | ```css
nav a:active {
    color: green;
}
``` |
